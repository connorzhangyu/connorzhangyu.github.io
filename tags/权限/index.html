解压缩

demo(文件夹)

|——|1.txt(文件)

# -r：递归地压缩目录，意味着 zip 命令会包含 demo 文件夹内的所有文件和子目录。
# demo.zip：这是输出的压缩文件名。
# demo：这是您要压缩的目录名。
zip -r demo.zip demo

# -o：强制覆盖现有的文件和文件夹，而不会询问用户。
# demo.zip：这是您想要解压的 .zip 文件名。
unzip -o demo.zip
