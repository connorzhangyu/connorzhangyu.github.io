    运行容器

    • docker run 运行容器的命令
    • -i 以交互模式运行,通常与-t一起使用
    • -t 为容器重新分配一个伪输入终端,通常与-i一起使用
    • bash 进入交互式终端,通常使用/bin/bash
    • -p 指定端口
    • -P 随机分配端口
    • -d 后台运行容器并返回容器Id,即启动守护式容器
