安装Vue

局部安装

局部安装并使用vue-cli 4.x版本,和创建项目

# 先创建一个文件夹
# 初始化一个node项目
npm init -y

# 局部安装vue
npm i -D @vue/cli

# 查看vue的版本
# 局部安装要用npx命令
C:\Users\mmzcg\WebstormProjects>npx vue -V
@vue/cli 5.0.8

# 创建一个项目
# 为啥不能用npm运行
npx vue create project-one

# 进入项目并且运行
cd project-one
npm run serve

# 也可以用ui
npx vue ui
